# Expert Domain — 넥서스 전문가 매칭

## 목적
11인의 박사 전문가 풀(넥서스)을 포함한 분야별 고수를 기업 수요에 맞게 매칭하고,
AI 결과물 검수 및 컨설팅 연결을 통해 플랫폼 신뢰도를 확보한다.

---

## 전문가 Pool 구성

### 넥서스 코어 (11인 박사)
- 역할: AI 생성 결과물(KPI, 계획서) 검수 및 보증
- 분야: 소재, 기계, 전자, 화학, 에너지, 바이오, ICT 등
- 등급: 넥서스 인증 (최고 신뢰 등급)

### 일반 전문가 Pool
- 역할: 과제 수행 단계 컨설팅, 기술 자문
- 분야: 특허, 인증, 사업화, 기술이전
- 등급: 평판 시스템으로 등급화

---

## 매칭 로직

### 매칭 기준 (우선순위 순)
1. 기술 분야 일치 (KSIC 업종 코드 매핑)
2. 지원사업 유형 경험 보유 여부
3. 과거 매칭 평점 (4.0 이상 우선)
4. 지역 근접성 (대면 가능 시)

### 매칭 트리거 시점
- KPI 생성 후 → 물리 검증 전문가 자동 추천
- 계획서 초안 생성 후 → 해당 분야 컨설턴트 추천
- 선정 확률 60% 미만 → 긴급 전문가 연결 제안

## 평판 시스템

| 등급 | 기준 | 배지 |
|------|------|------|
| 넥서스 인증 | 박사 + 내부 검증 | 🔵 |
| 골드 | 평점 4.5 이상 + 10건 | 🟡 |
| 실버 | 평점 4.0 이상 + 5건 | ⚪ |
| 일반 | 등록 완료 | - |

## Claude 작업 규칙

- 전문가 매칭 알고리즘 수정 시 matching_strategy.md 업데이트
- 새로운 분야 추가 시 분야 코드 테이블 업데이트
- 넥서스 코어 11인 정보는 하드코딩 (별도 관리)
- 매칭 결과는 이유와 함께 출력 (투명성 확보)
- MVP 단계에서는 수동 매칭도 허용 (자동화 완성 전)
